How to Use Quanty

Installation

Clone the repository:

git clone https://github.com/lopushok9/gemini_quant.git
cd gemini_quant

Starting the CLI

launch the terminal interface, in ai-investor folder

cd ai-investor
gemini extension install

Analyzing Markets

Run gemini analyze. You can choose different modes to get a comprehensive report on your target asset.
